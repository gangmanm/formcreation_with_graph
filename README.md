# 그래프를 이용한 설문지 만들기 페이지 
###사이트 URL
https://gangmanm.github.io/formcreation_with_graph/


간단한 설문지를 제작할 수 있는 페이지 입니다. 

- [X] 체크박스, 객관식, 서술형 3가지 유형의 질문생성
- [X] 질문 옵션 추가, 편집
- [X] 부모 질문에서 분기 설정 가능
- [X] 그래프를 통한 질문 flow 확인 가능
- [X] 드래그 앤 드롭으로 질문 순서 변경
- [X] 드래그 앤 드롭으로 그래프 모양 변경
- [X] 그래프 이미지 저장
